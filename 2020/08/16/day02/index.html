#####4. python的函数 基础入门

函数的定义：

shell:
    func test(){
        函数体
    }
    test() ### 函数的调用

    def test():
        函数体
    test() ### 函数调用
  • 参数的分类
    • 位置参数
    • 关键字参数

参数组合：

第二种：
动态接收位置参数， 动态接收关键字参数配合起来使用
