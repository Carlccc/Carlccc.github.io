1. 编译型语言和解释型语言的区别

编译型语言(强类型语言) : 一次编译，到处运行 例：c, c++, java等
解释型语言(弱类型语言)：不需要编译，直接调用解释器执行 例：php，python， shell

解释型语言修改完后，不需要编译，执行流程是从上到下依次解释执行。shell

4. python的变量
x = 1+2+3+4+5  
y = x*5 + 6
z = y*7
print(z)

变量就是将中间的结果暂存到内存,以便后续程序调用

类比shell， 和shell中的变量的定义是一致的。
但是和shell中不一样的地方的是：shell的等号两边不允许有空格，但是python等号两边也可以没有空格。那为啥必须要有空格？

  • 从视觉上会好看一些，调试代码比较舒服
  • 写代码，需要遵循 PEP8 规范
    • 变量的等号两边必须是有空格的，各大公司code review
    • 变量名必须要有意义 shell
    • 变量名只能是 字母、数字或下划线的任意组合
    • 变量名建议不使用拼音和中文
    • 变量的要具有意义
    • 变量名不要过长

编程语言的三种风格

  • 大驼峰法 单词的首字母一定是大写的。

    UserName, GetInfo, 等

  • 小驼峰法 单词的首字母小写，其余大写

    userName

  • 下划线法 单词和单词之间使用下划线拼接

    user_name

在代码 code review (代码评审) , 我们要求三种风格你可以随便选择一种你喜欢的风格，但是大家在选择完成之后，一定要主要的是，千万别中途换风格。

7.2 循环控制
  • while 循环
### 基本款
while 条件:
    # 循环体
    # 如果条件为真，那么循环则执行
    # 如果条件为假，那么循环不执行

例子：
num = int(input('请输入数字:'))  ## 12

while num > 0: ### 每次循环num和0会进行比较，没如果为True,则执行代码块中的内容
    print('dsadsadasdsasa')
    num = num - 1
else:   #### 循环退出之后，都会执行else中的代码
    print('last line.....')

两大关键字：
break 和 continue：

  • break： 退出整个循环，循环已经退出
  • continue: 跳出本次循环， 循环并没有退出


#### break ： 当警察抓到犯人的时候，后面即便有人，也不会做任何操作，直接退出
